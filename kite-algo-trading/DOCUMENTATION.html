🎯 Purpose

A comprehensive algorithmic trading platform designed for spot trading with automatic detection of sudden price movements in stocks. The application provides real-time market data access, order management, strategy execution, and backtesting capabilities.

✨ Key Features

📊 Real-time Market Data

Live quotes and historical data via Kite Connect API with WebSocket support for real-time updates.

📈 Order Management

Complete order lifecycle management with place, modify, cancel operations and risk controls.

🎯 Sudden Movement Strategy

Automated detection of sudden price movements (>2% in 5 minutes) with technical indicators.

🧪 Backtesting Engine

Test strategies on historical data with detailed performance metrics and analysis.

🖥️ Web Dashboard

Interactive interface for monitoring, control, and real-time performance tracking.

💾 PostgreSQL Persistence

Robust data storage with JPA entities for orders, trades, strategies, and market data.

Table Descriptions

📋 Orders Table

Stores all trading orders with complete lifecycle tracking.

Column Type Description Constraints
id bigint Primary key Auto-increment
order_id varchar(255) Unique order identifier UNIQUE, NOT NULL
symbol varchar(255) Trading symbol NOT NULL, INDEXED
order_type varchar(255) Order type (MARKET, LIMIT, etc.) NOT NULL, CHECK constraint
transaction_type varchar(255) BUY or SELL NOT NULL, CHECK constraint
quantity integer Order quantity NOT NULL
price numeric(38,2) Order price NULLABLE
status varchar(255) Order status NOT NULL, CHECK constraint, INDEXED
paper_trading boolean Paper trading flag NOT NULL
created_at timestamp Order creation time NOT NULL
updated_at timestamp Last update time NOT NULL

📊 Trades Table

Records completed trades with performance metrics.

Column Type Description
id bigint Primary key
trade_id varchar(255) Unique trade identifier
symbol varchar(255) Trading symbol
trade_type varchar(255) LONG or SHORT
entry_price numeric(19,4) Entry price
exit_price numeric(19,4) Exit price
pnl numeric(19,4) Profit/Loss amount
pnl_percentage numeric(5,2) P&L percentage
status varchar(255) Trade status
strategy_name varchar(255) Strategy that generated trade

Stores trading strategy configurations and performance metrics.

Column Type Description

is_active boolean Active status
total_trades integer Total trades executed
total_pnl numeric(19,4) Total profit/loss
win_rate numeric(5,2) Win rate percentage

            MarketData marketData = convertQuoteToMarketData(
                    entry.getKey(), entry.getValue());
            marketDataRepository.save(marketData);
        }
    }

Core Strategy Code

@Component
public class SuddenMovementStrategy {

    @Value("${strategy.sudden-movement.threshold:2.0}")
    private double movementThreshold;

    public List detectSuddenMovements(
            String symbol, List prices, List volumes) {

        List movements = new ArrayList<>();

        for (int i = 1; i < prices.size(); i++) {
            BigDecimal currentPrice = prices.get(i);
            BigDecimal previousPrice = prices.get(i - 1);

            // Calculate percentage change
            BigDecimal percentageChange = calculatePercentageChange(
                    previousPrice, currentPrice);

            // Check if movement exceeds threshold
            if (Math.abs(percentageChange.doubleValue()) >= movementThreshold) {
                PriceMovement movement = createPriceMovement(
                        symbol, previousPrice, currentPrice,
                        percentageChange, volumes.get(i),
                        determineMovementType(percentageChange));
                movements.add(movement);
            }
        }

        return movements;
    }

    public BigDecimal calculateRSI(List prices, int period) {
        // Calculate RSI using standard formula
        List gains = new ArrayList<>();
        List losses = new ArrayList<>();

        for (int i = 1; i < prices.size(); i++) {
            BigDecimal change = prices.get(i).subtract(prices.get(i - 1));
            if (change.compareTo(BigDecimal.ZERO) > 0) {
                gains.add(change);
                losses.add(BigDecimal.ZERO);
            } else {
                gains.add(BigDecimal.ZERO);
                losses.add(change.abs());
            }
        }

        BigDecimal avgGain = calculateAverage(
                gains.subList(gains.size() - period, gains.size()));
        BigDecimal avgLoss = calculateAverage(
                losses.subList(losses.size() - period, losses.size()));

        if (avgLoss.compareTo(BigDecimal.ZERO) == 0) {
            return BigDecimal.valueOf(100);
        }

        BigDecimal rs = avgGain.divide(avgLoss, 4, RoundingMode.HALF_UP);
        BigDecimal rsi = BigDecimal.valueOf(100).subtract(
                BigDecimal.valueOf(100).divide(BigDecimal.ONE.add(rs),
                        2, RoundingMode.HALF_UP));

        return rsi;
    }

    public TradingSignal generateTradingSignal(
            PriceMovement movement, BigDecimal rsi, MACDResult macd) {

        TradingSignal signal = new TradingSignal();
        signal.setSymbol(movement.getSymbol());
        signal.setMovementType(movement.getMovementType());

        if (movement.getMovementType() == MovementType.SUDDEN_RISE) {
            // Buy signal conditions
            if (rsi.compareTo(BigDecimal.valueOf(70)) < 0 &&
                macd.getHistogram().compareTo(BigDecimal.ZERO) > 0) {
                signal.setSignalType(SignalType.BUY);
                signal.setConfidence(calculateConfidence(movement, rsi, macd, true));
            } else {
                signal.setSignalType(SignalType.HOLD);
            }
        } else if (movement.getMovementType() == MovementType.SUDDEN_FALL) {
            // Sell signal conditions
            if (rsi.compareTo(BigDecimal.valueOf(30)) > 0 &&
                macd.getHistogram().compareTo(BigDecimal.ZERO) < 0) {
                signal.setSignalType(SignalType.SELL);
                signal.setConfidence(calculateConfidence(movement, rsi, macd, false));
            } else {
                signal.setSignalType(SignalType.HOLD);
            }
        }

        return signal;
    }
}

Strategy Execution Engine

@Component
public class StrategyEngine {

    @Scheduled(fixedRateString = "${strategy.execution.interval:60000}")
    public void executeStrategies() {
        List activeStrategies = strategyRepository.findByIsActiveTrue();

        for (Strategy strategy : activeStrategies) {
            switch (strategy.getType()) {
                case SUDDEN_MOVEMENT:
                    executeSuddenMovementStrategy(strategy);
                    break;
                // ... other strategy types
            }
        }
    }

    private void executeSuddenMovementStrategy(Strategy strategy) {
        List watchedSymbols = getWatchedSymbols(strategy);

        for (String symbol : watchedSymbols) {
            // Get recent price data
            List recentPrices = getRecentPrices(symbol, 10);
            List recentVolumes = getRecentVolumes(symbol, 10);

            // Detect sudden movements
            List movements =
                    suddenMovementStrategy.detectSuddenMovements(
                            symbol, recentPrices, recentVolumes);

            // Process each movement
            for (PriceMovement movement : movements) {
                processMovement(movement, strategy);
            }
        }
    }

    @Scheduled(fixedRate = 30000)
    public void monitorOpenPositions() {
        // Check exit conditions every 30 seconds
        checkExitConditions();
    }

    private void checkExitConditions() {
        List openTrades = getOpenTrades();

        for (Trade trade : openTrades) {
            BigDecimal currentPrice = getCurrentPrice(trade.getSymbol());

            if (shouldExitTrade(trade, currentPrice)) {
                closeTrade(trade, currentPrice);
            }
        }
    }
}
